\begin{frame}[label=frame_1]{Задача}
\begin{itemize}
\item Инструмент для моделирования уравнений с запаздыванием, распределенных в одно-, дву- и трехмерных областях (содержащих диффузионный член)
\item Поддержка современного оборудования --- неоднородные архитектуры с иерархической организацией памяти
\item Повышение уровня абстракции --- автоматическая генерация шаблонного кода для пользовательских функций
\item Интерфейс для управления вычислениями
\end{itemize}
\end{frame}

\begin{frame}[label=frame_1]{Существующие проекты}
Моделирование уравнений в частных производных (уравнения с запаздыванием не поддерживаются)
\begin{itemize}
\item PETSc --- Portable, Extensible Toolkit for
Scientific Computation
\item DUNE --- Distributed and Unified Numerics Environment
\item HYPRE --- High Performance Preconditioners
\item ExaStencils (в разработке)
\end{itemize}

\end{frame}



\begin{frame}[label=frame_2]{Структура пакета}
\begin{itemize}
\item Пользовательский интерфейс 
\begin{itemize}
\item создание/модификации проекта
\item управление вычислениями
\end{itemize}
\item Предварительная обработка
\begin{itemize}
\item геометрия и свойства области
\item библиотека пользовательских функций
\item разбиение задачи на блоки и распределение блоков по вычислительным устройствам
\end{itemize}
\item Вычислительное ядро
\begin{itemize}
\item параллельные вычисления
\item коммуникация с интерфейсом 
 \end{itemize}

\end{itemize}
\end{frame}



\begin{frame}[label=frame_3]{Вычислительное ядро}
\begin{itemize}
\item Параллельный фреймворк
\begin{itemize}
\item крупнозернистый параллелизм: разбиение задачи на блоки и распределение блоков по устройствам
\item пересылка необходимых данных между устройствами на разных узлах с помощью MPI 
\item обмен данными через pinned-память между устройствами на одном узле
\item мелкозернистый параллелизм в рамках блока: работа с общей памятью, OpenMP на CPU и CUDA на GPU
\item использование арифметики двойной точности: CPU дает значительный вклад в общую производительность
\end{itemize}
\item Алгоритмы интегрирования и анализа данных
\begin{itemize}
\item Явные схемы Рунге-Кутты с контролем точности на шаге
\item Алгоритмы продолжения по параметру для поиска фазовых перестроек
\end{itemize}
\end{itemize}

\end{frame}

\begin{frame}[label=frame_222]{Тестирование производительности}
\begin{itemize}
\item Оборудование: 2 узла: 2xCPU E5-2690 (8 ядер, 2.9ГГц), 3хGPU Tesla M2090, Infiniband QDR 40Gbps

\item ПО: SLES 12, gcc 4.8,  Mellanox OFED 2.4, OpenMPI 1.8, CUDA Toolkit 7.0, SLURM 14.11 

\item Тестовая задача: уравнение теплопроводности

\item Метод Эйлера, постоянный шаг

\end{itemize}
В дальнейшем 2 центральных процессора (16 ядер) в рамках одного узла используются как единое вычислительное устройство, обозначеное CPU.
\end{frame}
